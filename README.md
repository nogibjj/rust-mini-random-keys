# Random Keys Generator in Rust CLI 
This is my week 2 Rust mini project, a random keys generator in Rust CLI.

A sample usage:
```
$ cargo run
```

```
Welcome to the Random Keys Generator in Rust!

Generated key: 9dRwgVszWTHGY6hFPYDqaIqB2gpESGWk
```
